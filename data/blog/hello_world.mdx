# 你好，世界

## 使用 C 语言输出 Hello World!

以下代码如何使用 C 语言输出 Hello World!

```C
// 引入头文件
#include <stdio.h>

// 主函数
int main(void) {
  printf("Hello World!");
  return 0;                 // 返回 0 表示程序成功执行
}
// Hello World!
```
